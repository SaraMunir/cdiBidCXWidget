On Demand Auctions Implementation

bidcx.com

Notes

  1. You’ll likely have to edit the path to the cd-ondemand directory when you include css and js files.

  2. Most of the following should work out-of-the-box, with minimal configuration. What you need to know:

    • The target auction site URL e.g. https://bidcx.com/ included in the parameters when launching the plugin
    • The target auction site id e.g. 1104483 included in the parameters when launching the plugin
    • Your siteâ€™s main branding colours (as in CSS), used for heading and button backgrounds applied to cd-ondemand-title elements and included in the parameters when launching the plugin

Include the ondemand CSS in your <head>:

<!-- Add ondemand CSS after main file so we can use that as a base -->
<link rel="stylesheet" href="cd-ondemand/dist/cd-ondemand.css">

Add the following code where you want to place the plugin. 

<div class="cd-ondemand-container">
  <div class="cd-ondemand" id="cd-ondemand-bidcx"></div>
</div>

Before the closing <body> tag, include the on-demand scripts:

<!-- cd-ondemand scripts -->
<script type="text/javascript" src="https://code.jquery.com/jquery.min.js"></script>  <!-- Only needed if jquery is not present in the site -->   
<script src="cd-ondemand/dist/cd-ondemand-min.js"></script>
<script src="cd-ondemand/dist/cd-ondemand-templates-min.js"></script>

… then launch on-demand:

<script>
(function($) {

  $('#cd-ondemand-bidcx').ondemand({
    site: '[auction site URL (no trailing /)]',
    siteId: [auction site id],
    primaryColor: '[Primary color]',
    secondaryColor: '[Secondary color]',
    background: '[Background color]',
    titleWidget: "Title Goes Here",
    parameters: 'pageName=partners&pgmode1=[Cause shortcode]&pgmode2=partner',
    items: 3,
    liveUpdate: false,
    utmString: '&utm_source=osuStore&utm_medium=ondemand&utm_campaign=bidcxOndemand&isyntrk-mktcampnm=nnnNflbidcxWidgBid',
    template: 'widgetTemplate',
    countdown: true
  });


})( jQuery );

</script>

Parameters that can be changed according to use case are explained as follows:

  • site : The target auction site URL e.g. https://bidcx.com/
  • siteId : The target auction site id e.g. 1104483
  • primaryColor : Gives the color to widget title, auction title, current bid and bid now button text.
  • secondaryColor : Gives the color to counter, carousel Indicators and bid now button background.
  • background : Responsible for the background color of the widget.
  • titleWidget : Sets the title of the widget.
  • Parameters: Requires you to input short code of the cause to be displayed in widget.
  • Items: No of products to be fetched.
